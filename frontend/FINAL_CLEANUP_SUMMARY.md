## 🔧 使用方法

### 基础操作
1. **访问应用**: http://localhost:5173/
2. **上传视频**: 拖拽视频文件到时间轴
3. **缩放操作**: Alt + 鼠标滚轮
4. **滚动操作**: Shift + 鼠标滚轮
5. **播放头操作**: 拖拽红色圆形手柄

### 精确编辑
1. **放大到帧级别**: 持续使用 Alt + 滚轮向上
2. **观察帧网格**: 网格线显示帧边界
3. **精确定位**: 播放头自动对齐到帧边界
4. **帧级编辑**: 所有操作都有帧精度
