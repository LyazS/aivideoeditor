# AI视频编辑器 - 文档总览

## 项目简介

AI视频编辑器是一个基于现代Web技术构建的视频编辑应用，采用Vue 3 + TypeScript + WebAV SDK技术栈，提供直观的拖拽式多轨道视频编辑体验。

## 核心特性

### 🎬 专业视频编辑
- **多轨道编辑**: 支持多个视频轨道同时编辑
- **精确控制**: 帧级精度的时间控制和位置调整
- **实时预览**: 基于WebAV SDK的硬件加速渲染
- **丰富变换**: 位置、缩放、旋转、透明度等属性调整

### 🎯 直观用户体验
- **拖拽操作**: 从素材库直接拖拽到时间轴
- **可视化编辑**: 所见即所得的编辑体验
- **响应式布局**: 可调整的面板布局
- **快捷操作**: 丰富的右键菜单和工具栏

### ⚡ 高性能渲染
- **WebCodecs API**: 利用浏览器原生硬件加速
- **智能缓存**: 优化的帧缓存和资源管理
- **流畅播放**: 多种播放速度和精确时间控制
- **内存优化**: 高效的资源管理和垃圾回收

### 🔧 现代技术栈
- **Vue 3**: Composition API + TypeScript
- **WebAV SDK**: 专业的Web视频处理引擎
- **Pinia**: 现代化的状态管理
- **Vite**: 快速的开发和构建工具

## 快速开始

### 环境要求
- **浏览器**: Chrome 94+ 或 Edge 94+ (需要WebCodecs API支持)
- **Node.js**: 16+ 版本
- **内存**: 建议8GB以上

### 安装运行
```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

### 基本使用
1. **导入视频**: 点击素材库的"+"按钮或拖拽文件
2. **添加到时间轴**: 从素材库拖拽视频到时间轴轨道
3. **编辑片段**: 选中片段后在属性面板调整属性
4. **预览播放**: 使用播放控制进行预览

## 性能建议

### 硬件要求
- **CPU**: 支持硬件视频解码的现代处理器
- **内存**: 8GB以上推荐
- **显卡**: 支持硬件加速的独立显卡（可选）

### 使用建议
- 使用较小的视频文件进行测试
- 避免同时打开过多浏览器标签
- 定期清理浏览器缓存
- 关闭不必要的浏览器扩展
